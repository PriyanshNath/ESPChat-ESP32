# 💬 ESPChat

ESPChat is an offline real-time chat application running on an ESP32.

## Features

- 🌐 ESP32 Hotspot
- 💬 Real-time WebSocket chat
- 💾 LittleFS storage
- 💡 LED notification
- 📱 Mobile friendly
- 🎨 Discord-inspired UI
- 💾 Chat history

## Hardware

- ESP32 Dev Module

## Libraries

- WebServer
- WebSocketsServer
- LittleFS
- ArduinoJson
